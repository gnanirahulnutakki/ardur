Ardur's public rule is that claims should point to code, a spec, a verifier
path, a media artifact, or a named limitation.

The current rerunnable Phase 1 proof path is the no-key source-checkout bundle:
run the Claude Code MVP quickstart, inspect `bundle.redacted.json`, then use the
demo packet to attach only the claims that bundle supports.
