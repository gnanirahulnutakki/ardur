# Code Of Conduct

Ardur is an engineering project that values direct, evidence-based
collaboration.

## Expected behavior

- Be respectful and technically honest.
- Challenge ideas with evidence, not personal attacks.
- Keep discussions focused on correctness, security, operability, and clear
  tradeoffs.
- Respect maintainers, reviewers, and other contributors' time.

## Unacceptable behavior

- harassment or discrimination
- personal insults or intimidation
- deliberately misleading technical claims
- publishing private security details before coordinated disclosure
- spamming, trolling, or derailing review threads

## Enforcement

Project maintainers may edit, moderate, lock, or remove content that violates
this code of conduct. Repeated or severe violations may lead to temporary or
permanent bans from project spaces.

## Reporting

For behavior issues, contact the maintainers through the project's published
community or repository contact path. For security issues, use `SECURITY.md`
instead.
